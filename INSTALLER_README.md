# Honeycutt Budget Planner - Installation

## What's Included

- **Honeycutt Budget Planner Setup 1.0.0.exe** - Full installer with desktop shortcuts
- **Honeycutt Budget Planner-1.0.0-portable.exe** - Portable version (no installation required)

## Installation Options

### Option 1: Full Installer (Recommended)
1. Run `Honeycutt Budget Planner Setup 1.0.0.exe`
2. Follow the installation wizard
3. Choose installation directory (default: Program Files)
4. Desktop and Start Menu shortcuts will be created automatically

### Option 2: Portable Version
1. Download `Honeycutt Budget Planner-1.0.0-portable.exe`
2. Place it in any folder
3. Double-click to run - no installation required
4. All data will be stored in the same folder

## System Requirements
- Windows 10 or later
- 64-bit system
- No internet connection required after installation
- No additional dependencies needed

## Data Storage
- Your budget data is stored locally on your computer
- No internet connection or cloud services required
- Data persists between application launches

## Uninstallation
For the installed version, use "Add or Remove Programs" in Windows Settings or the uninstaller included with the application.

The portable version can be removed by simply deleting the executable file.
